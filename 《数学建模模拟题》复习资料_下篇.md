## 第8题（模型表示第1题）：药剂生产规划（线性规划）

### 原题

某药剂厂生产两种药剂 $a$ 和 $b$。

- 生产1kg $a$：原料 $c$ 3kg、原料 $d$ 4kg、工时5
- 生产1kg $b$：原料 $d$ 2kg、原料 $e$ 5kg、工时7

价格与成本：
- 售价：$a$ 1000元/kg，$b$ 800元/kg
- 原料单价：$c$ 30元/kg，$d$ 40元/kg，$e$ 50元/kg
- 工时费：25元/工时

资源上限：
- $c \le 300$kg，$d \le 400$kg，$e \le 350$kg
- 工时 $\le 600$

建立线性规划模型，使每周利润最大。

### 知识点补充

**核心知识点：线性规划模型的建立**

#### 1. 什么是线性规划？

线性规划是数学建模中最基础、最重要的方法之一。它解决的是"在有限资源下，如何安排生产使利润最大（或成本最小）"这类问题。

线性规划的三要素：
- **决策变量**：你要决定什么（比如生产多少 kg 的 a 和 b）
- **目标函数**：你要优化什么（比如总利润）
- **约束条件**：你受到哪些限制（比如原料有限、工时有限）

"线性"的含义：目标函数和约束条件都是**一次**的（变量只有一次方，没有 $x^2$、$\sqrt{x}$ 等）。

#### 2. 如何确定决策变量？

决策变量是我们可以主动改变的量。本题中，我们需要决定每周各生产多少药剂的量。

设：
- $x_1$：药剂 $a$ 的每周生产量（单位：kg）
- $x_2$：药剂 $b$ 的每周生产量（单位：kg）

**注意**：变量通常用带有下标的字母表示，如 $x_1, x_2$。在书写时 $x_1$ 表示 $x$ 下标 $1$。

#### 3. 如何计算利润？

利润 = 收入 - 成本

**收入**：卖产品得到的钱
- 卖 $a$：$1000x_1$ 元
- 卖 $b$：$800x_2$ 元
- 总收入 = $1000x_1 + 800x_2$

**成本**：需要算清楚每一种原料和工时的花费

原料 $c$ 的成本：生产 $a$ 用到 $c$，每kg $a$ 需要 3kg $c$，生产 $x_1$ kg $a$ 需要 $3x_1$ kg $c$。$c$ 单价 30 元/kg，所以原料 $c$ 成本 = $30 \times 3x_1 = 90x_1$

同理：
- 原料 $d$ 成本：$a$ 用 4kg $d$ 每kg，$b$ 用 2kg $d$ 每kg → $40 \times (4x_1 + 2x_2) = 160x_1 + 80x_2$
- 原料 $e$ 成本：$b$ 用 5kg $e$ 每kg → $50 \times 5x_2 = 250x_2$
- 工时成本：$a$ 需 5 工时/kg，$b$ 需 7 工时/kg → $25 \times (5x_1 + 7x_2) = 125x_1 + 175x_2$

总成本 = $(90 + 160 + 125)x_1 + (80 + 250 + 175)x_2 = 375x_1 + 505x_2$

总利润 = 总收入 - 总成本 = $(1000x_1 + 800x_2) - (375x_1 + 505x_2) = 625x_1 + 295x_2$

#### 4. 如何确定约束条件？

约束条件就是资源的上限。用掉的资源不能超过可用总量。

**原料 $c$ 的限制**：$a$ 消耗 $c$，$b$ 不消耗 $c$
$3x_1 \le 300$

**原料 $d$ 的限制**：$a$ 和 $b$ 都消耗 $d$
$4x_1 + 2x_2 \le 400$

**原料 $e$ 的限制**：只有 $b$ 消耗 $e$
$5x_2 \le 350$

**工时的限制**：$a$ 和 $b$ 都消耗工时
$5x_1 + 7x_2 \le 600$

**非负约束**：生产量不能为负数
$x_1 \ge 0, \quad x_2 \ge 0$

### 题目讲解

**第1步：确定决策变量**

设 $x_1$ = 药剂 $a$ 的周产量（kg），$x_2$ = 药剂 $b$ 的周产量（kg）。

**为什么这么做？** 决策变量是生产规划问题最直接的选择——我们需要决定的就是每种药生产多少。

**第2步：写出目标函数**

$$\max z = 625x_1 + 295x_2$$

其中 $z$ 表示总利润。系数 625 是每kg $a$ 的利润（收入1000 - 原料成本 90-160-125），295 是每kg $b$ 的利润。

**为什么这么做？** 目标函数 = 收入 - 成本。把前面算好的利润表达式写出来即可。

**第3步：写出约束条件**

$$
\begin{cases}
3x_1 \le 300 & \text{（原料 c 上限）} \\
4x_1 + 2x_2 \le 400 & \text{（原料 d 上限）} \\
5x_2 \le 350 & \text{（原料 e 上限）} \\
5x_1 + 7x_2 \le 600 & \text{（工时上限）} \\
x_1 \ge 0, \; x_2 \ge 0 & \text{（非负约束）}
\end{cases}
$$

**为什么这么做？** 逐条把题目中给出的资源限制翻译成数学不等式。"用掉的 ≤ 可用的"是线性规划约束的标准格式。

**标准答案：**

决策变量：$x_1$ = 药剂 $a$ 的周产量（kg），$x_2$ = 药剂 $b$ 的周产量（kg）

目标函数：
$$\max z = 625x_1 + 295x_2$$

约束条件：
$$
\begin{cases}
3x_1 \le 300 \\
4x_1 + 2x_2 \le 400 \\
5x_2 \le 350 \\
5x_1 + 7x_2 \le 600 \\
x_1, x_2 \ge 0
\end{cases}
$$

## 第10题（模型表示第3题）：货机装运问题

### 原题

**表1 货舱限制**

|        | 前仓 | 中仓 | 后仓 |
|--------|------|------|------|
| 质量(t) | 10   | 16   | 8    |
| 体积(m³)| 6800 | 8700 | 5300 |

平衡要求：实际装载质量与最大容许质量成比例。

**表2 货物信息**

|        | 质量(t) | 体积(m³/t) | 利润(元/t) |
|--------|---------|------------|------------|
| 货物1  | 18      | 480        | 3100       |
| 货物2  | 15      | 650        | 3800       |
| 货物3  | 23      | 580        | 3500       |
| 货物4  | 12      | 390        | 2850       |

建立模型，使总利润最大（不需求解）。

### 知识点补充

**核心知识点：多仓多货物流配载模型 + 比例平衡约束**

#### 1. 题目背景

这是一道**多仓配载**问题。飞机有三个货舱（前、中、后），每个货舱有质量上限和体积上限。有四种货物，每种货物的总可用量是有限的，且单位体积和单位利润不同。

除了正常的容量限制外，还有一个**平衡要求**：每个货舱的实际装载质量与最大容许质量的**比例**必须相等。

#### 2. 什么是"比例平衡"？

"实际装载质量与最大容许质量成比例"的意思是：

$$\frac{\text{前仓实际装载质量}}{\text{前仓最大容许质量}} = \frac{\text{中仓实际装载质量}}{\text{中仓最大容许质量}} = \frac{\text{后仓实际装载质量}}{\text{后仓最大容许质量}}$$

即三个货舱的"装填比例"完全相等。设这个共同比例为 $r$，则：

前仓质量 = $10r$，中仓质量 = $16r$，后仓质量 = $8r$

其中 $0 \le r \le 1$（不能超过最大容许质量）。

#### 3. 决策变量的设定

这道题的关键是：不仅要决定运多少货物，还要决定每种货物放在哪个货舱。所以需要**双下标变量**。

设 $x_{ij}$ 表示第 $i$ 种货物放到第 $j$ 个货舱的质量（吨）。

- $i = 1,2,3,4$：四种货物
- $j = 1,2,3$：前、中、后三个货舱

例如 $x_{11}$ 表示货物1放到前仓的质量（吨）。

#### 4. 约束的分类

**货物可用量约束**：每种货物的总装载量不超过它的可用量。
- 货物1总量：$x_{11} + x_{12} + x_{13} \le 18$
- 货物2总量：$x_{21} + x_{22} + x_{23} \le 15$
- 货物3总量：$x_{31} + x_{32} + x_{33} \le 23$
- 货物4总量：$x_{41} + x_{42} + x_{43} \le 12$

**货舱质量约束**：每个货舱装载的总质量不超过其最大容许质量。
- 前仓：$x_{11} + x_{21} + x_{31} + x_{41} \le 10$
- 中仓：$x_{12} + x_{22} + x_{32} + x_{42} \le 16$
- 后仓：$x_{13} + x_{23} + x_{33} + x_{43} \le 8$

**货舱体积约束**：每种货物有单位体积，每个货舱的总体积不超过其容量。
- 前仓：$480x_{11} + 650x_{21} + 580x_{31} + 390x_{41} \le 6800$
- 中仓：$480x_{12} + 650x_{22} + 580x_{32} + 390x_{42} \le 8700$
- 后仓：$480x_{13} + 650x_{23} + 580x_{33} + 390x_{43} \le 5300$

**平衡约束**：三个货舱的装载比例相等。
$$\frac{x_{11} + x_{21} + x_{31} + x_{41}}{10} = \frac{x_{12} + x_{22} + x_{32} + x_{42}}{16} = \frac{x_{13} + x_{23} + x_{33} + x_{43}}{8}$$

可以拆成两个等式：
$$\frac{x_{11} + x_{21} + x_{31} + x_{41}}{10} = \frac{x_{12} + x_{22} + x_{32} + x_{42}}{16}$$
$$\frac{x_{11} + x_{21} + x_{31} + x_{41}}{10} = \frac{x_{13} + x_{23} + x_{33} + x_{43}}{8}$$

#### 5. 目标函数的写法

总利润 = 每种货物的装载量 × 该货物的单位利润。

$$\max Z = 3100(x_{11}+x_{12}+x_{13}) + 3800(x_{21}+x_{22}+x_{23}) + 3500(x_{31}+x_{32}+x_{33}) + 2850(x_{41}+x_{42}+x_{43})$$

### 题目讲解

**第1步：确定决策变量**

设 $x_{ij}$ 为第 $i$ 种货物放入第 $j$ 个货舱的质量（吨），$i = 1,2,3,4$，$j = 1,2,3$。

**为什么这么做？** 双下标变量可以区分"哪种货"和"放在哪里"两个维度，这是多仓配载问题的标准做法。

**第2步：写出目标函数**

$$\max Z = \sum_{i=1}^{4} \sum_{j=1}^{3} p_i x_{ij}$$

其中 $p_1 = 3100$，$p_2 = 3800$，$p_3 = 3500$，$p_4 = 2850$ 分别是四种货物的单位利润。

**为什么这么做？** 每种货物的总装载量乘以它的单位利润，再求和就是总利润。

**第3步：写出约束条件**

**货物可用量约束**：
$$
\begin{cases}
x_{11} + x_{12} + x_{13} \le 18 \\
x_{21} + x_{22} + x_{23} \le 15 \\
x_{31} + x_{32} + x_{33} \le 23 \\
x_{41} + x_{42} + x_{43} \le 12
\end{cases}
$$

**货舱质量约束**：
$$
\begin{cases}
x_{11} + x_{21} + x_{31} + x_{41} \le 10 \\
x_{12} + x_{22} + x_{32} + x_{42} \le 16 \\
x_{13} + x_{23} + x_{33} + x_{43} \le 8
\end{cases}
$$

**货舱体积约束**：
$$
\begin{cases}
480x_{11} + 650x_{21} + 580x_{31} + 390x_{41} \le 6800 \\
480x_{12} + 650x_{22} + 580x_{32} + 390x_{42} \le 8700 \\
480x_{13} + 650x_{23} + 580x_{33} + 390x_{43} \le 5300
\end{cases}
$$

**平衡约束**：
$$\frac{x_{11}+x_{21}+x_{31}+x_{41}}{10} = \frac{x_{12}+x_{22}+x_{32}+x_{42}}{16}$$
$$\frac{x_{11}+x_{21}+x_{31}+x_{41}}{10} = \frac{x_{13}+x_{23}+x_{33}+x_{43}}{8}$$

**非负约束**：$x_{ij} \ge 0$，$i = 1,2,3,4$，$j = 1,2,3$

**为什么这么做？** 四种约束分别对应：货物是有限的不能超量使用、每个货舱有承重和体积限制、平衡要求三个货舱装载比例相等。

**标准答案：**

设 $x_{ij}$ 为第 $i$ 种货物放入第 $j$ 个货舱的质量（吨），$i=1,2,3,4$，$j=1,2,3$（分别对应前、中、后仓）。

$$\max Z = 3100\sum_{j=1}^{3}x_{1j} + 3800\sum_{j=1}^{3}x_{2j} + 3500\sum_{j=1}^{3}x_{3j} + 2850\sum_{j=1}^{3}x_{4j}$$

约束条件：
$$
\begin{cases}
\displaystyle\sum_{j=1}^{3}x_{1j} \le 18, \quad \sum_{j=1}^{3}x_{2j} \le 15, \quad \sum_{j=1}^{3}x_{3j} \le 23, \quad \sum_{j=1}^{3}x_{4j} \le 12 & \text{（货物可用量）} \\[6pt]
\displaystyle\sum_{i=1}^{4}x_{i1} \le 10, \quad \sum_{i=1}^{4}x_{i2} \le 16, \quad \sum_{i=1}^{4}x_{i3} \le 8 & \text{（货舱质量）} \\[6pt]
480x_{11}+650x_{21}+580x_{31}+390x_{41} \le 6800 & \text{（前仓体积）} \\
480x_{12}+650x_{22}+580x_{32}+390x_{42} \le 8700 & \text{（中仓体积）} \\
480x_{13}+650x_{23}+580x_{33}+390x_{43} \le 5300 & \text{（后仓体积）} \\[6pt]
\displaystyle\frac{\sum_{i=1}^{4}x_{i1}}{10} = \frac{\sum_{i=1}^{4}x_{i2}}{16} = \frac{\sum_{i=1}^{4}x_{i3}}{8} & \text{（平衡约束）} \\[6pt]
x_{ij} \ge 0, \quad i=1,2,3,4, \quad j=1,2,3
\end{cases}
$$

## 第9题（模型表示第2题）：发动机生产计划

### 原题

某厂需在第1、2、3季度末分别交货：40、60、80台。

- 每季度生产费用：$f(x) = ax + bx^2$，$x$ 为当季产量
- 剩余可储存，每台每季度储存费 $c$
- 每季度最大产能100台
- 期初无存货

建立模型，满足合同且总费用最小（不需求解）。

### 知识点补充

**核心知识点：多阶段生产-库存优化模型**

#### 1. 题目模型说明

这道题是一道典型的**多阶段决策问题**。我们需要决定每个季度生产多少台发动机，既要满足当季的交货需求，又要考虑存货成本，同时不超出产能限制。

题目中的 $a$、$b$、$c$ 都是已知常数（具体数值未给出），建立模型时直接用字母表示即可。

#### 2. 决策变量的确定

对于多阶段问题，通常为每个阶段设置一个**生产量**变量。

本题有三个季度，设有三个决策变量：
- $x_1$：第1季度产量
- $x_2$：第2季度产量
- $x_3$：第3季度产量

还要设置**库存量**变量（因为未卖出的可以储存）：
- $I_1$：第1季度末库存
- $I_2$：第2季度末库存
- $I_3$：第3季度末库存（期末应该为0，因为交货后就结束了）

或者，更简洁的做法是用生产量和需求量的关系隐式表达库存。

#### 3. 什么是库存平衡方程？

库存平衡方程描述了"上期库存 + 本期生产 - 本期交货 = 本期库存"的关系。

本题三个季度的需求（交货量）分别为：$d_1 = 40$，$d_2 = 60$，$d_3 = 80$。

库存平衡：
- 第1季度末库存：$I_1 = x_1 - 40$（期初无存货，所以 $I_0 = 0$）
- 第2季度末库存：$I_2 = I_1 + x_2 - 60$
- 第3季度末库存：$I_3 = I_2 + x_3 - 80 = 0$（期末无存货）

#### 4. 目标函数的构成

总费用 = 各季度的生产费用之和 + 各季度的库存费用之和

**生产费用**：第 $i$ 季度的生产费用 = $f(x_i) = a x_i + b x_i^2$

**库存费用**：第 $i$ 季度的库存费用 = $c \times I_i$（$I_i$ 是第 $i$ 季度末的库存量，$c$ 是每台每季度储存费）

总费用 = $\sum_{i=1}^{3} (a x_i + b x_i^2) + c \sum_{i=1}^{2} I_i$（第3季度末库存为0，没有库存费）

#### 5. 产能限制

每个季度最多生产100台：
$x_1 \le 100, \quad x_2 \le 100, \quad x_3 \le 100$

### 题目讲解

**第1步：确定决策变量**

设 $x_i$ 为第 $i$ 季度的发动机产量（台），$i = 1, 2, 3$。
设 $I_i$ 为第 $i$ 季度末的库存量（台），$i = 1, 2$。

**为什么这么做？** 每个季度的产量是我们可以主动调控的，库存量则由产量和需求的差值决定。

**第2步：写出库存平衡关系**

第1季度：$I_1 = x_1 - 40$
第2季度：$I_2 = I_1 + x_2 - 60$
第3季度：$x_3 - 80 \ge 0$（不允许缺货），且期末库存 $I_3 = I_2 + x_3 - 80 = 0$

**为什么这么做？** 用数学等式描述"期初库存 + 产量 - 交货量 = 期末库存"这个物流关系。

**第3步：写出目标函数**

$$\min Z = \sum_{i=1}^{3} (a x_i + b x_i^2) + c(I_1 + I_2)$$

**为什么这么做？** 总费用 = 所有季度的生产费 + 所有季度的储存费。这里 $I_1 + I_2$ 是因为第3季度末库存为0，不需要储存费。

**为什么要用 $\min$？** 因为我们要让总费用最小（成本最小化问题用 min，利润最大化问题用 max）。

**第4步：写出约束条件**

$$
\begin{cases}
x_1 \le 100, \; x_2 \le 100, \; x_3 \le 100 & \text{（产能限制）} \\
I_1 = x_1 - 40 & \text{（第1季度库存）} \\
I_2 = I_1 + x_2 - 60 & \text{（第2季度库存）} \\
I_2 + x_3 - 80 = 0 & \text{（第3季度库存为0）} \\
x_1, x_2, x_3 \ge 0, \; I_1, I_2 \ge 0 & \text{（非负约束）}
\end{cases}
$$

**为什么这么做？** 产能限制是题目直接给的。库存平衡关系把各季度的生产、库存、交货连接在一起。

**标准答案：**

决策变量：$x_1, x_2, x_3$（各季度产量），$I_1, I_2$（各季度末库存）

目标函数：
$$\min Z = \sum_{i=1}^{3} (a x_i + b x_i^2) + c(I_1 + I_2)$$

约束条件：
$$
\begin{cases}
0 \le x_i \le 100, \quad i = 1,2,3 \\
I_1 = x_1 - 40 \\
I_2 = I_1 + x_2 - 60 \\
I_2 + x_3 - 80 = 0 \\
I_1, I_2 \ge 0
\end{cases}
$$
