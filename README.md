# 期末周速成 SKILL

> 为 Trae AI IDE 设计的「从零到及格」理工科期末复习资料生成技能

## 简介

本项目是一个 [Trae AI IDE](https://www.trae.com.cn) 的自定义技能（Skill），旨在帮助零基础或基础薄弱的理工科学生在期末考试前快速复习。用户只需提供一份试卷，该技能即可逐题生成结构化的复习资料，涵盖：

- **原题呈现** - 原汁原味的题目内容
- **知识点补充** - 从零讲起，不给公式而是帮助建立直观理解
- **题目讲解** - 手把手带你分析题意、建立模型、求解并解读结果

## 快速开始

### 1. 安装技能

将本项目中的 `.trae/skills/` 目录复制到你的 Trae 项目根目录下：

```
你的项目目录/
└── .trae/
    └── skills/
        └── zero-to-pass/
            └── SKILL.md
```

### 2. 准备试卷

将需要复习的试卷整理成 Markdown（`.md`）或纯文本（`.txt`）格式，放入你的项目目录中。

> PDF 格式的试卷可先通过工具（如 `pdf2md`、`Marker` 等）转换为 Markdown 格式。

### 3. 调用技能

在 Trae AI IDE 中打开项目，输入提示词调用本技能：

```
@zero-to-pass 帮我复习 试卷文件.md
```

技能会自动逐题生成复习资料并写入 Markdown 文件中。
