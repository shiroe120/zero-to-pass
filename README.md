# 期末周速成 SKILL

> 「从零到及格」理工科期末复习资料生成技能

## 简介

这是一个自定义 AI 技能（Skill），旨在帮助零基础或基础薄弱的理工科学生在期末考试前快速复习。用户只需提供一份试卷，该技能即可逐题生成结构化的复习资料，涵盖：

- **原题呈现** - 原汁原味的题目内容
- **知识点补充** - 从零讲起，补充足够的前置知识，逐步延伸到本题所需深度
- **题目讲解** - 手把手带你结合上述知识点讲解，分析题意、建立模型、求解并解读结果

## 支持的 AI IDE

本技能兼容以下支持 Claude Skills 规范的 AI IDE：

| IDE | 安装路径 | 说明 |
|-----|---------|------|
| **Trae** | `.trae/skills/` | 国产 AI IDE，原生支持 |
| **Claude Code** | `.claude/skills/` | 官方 CLI 工具，规范源头 |
| **其他兼容 IDE** | 同上或类似路径 | 遵循 Skills 规范即可 |

## 快速开始

### 1. 安装技能

将本项目中的 skills 目录复制到你的项目根目录下：

```
你的项目/
└── .trae/              # Trae 用
    └── skills/
        └── zero-to-pass/
            └── SKILL.md
```

如果使用 **Claude Code**，目录名改为 `.claude`；其他 IDE 请参照其文档确定技能安装路径。

### 2. 准备试卷

将需要复习的试卷整理成 Markdown（`.md`）或纯文本（`.txt`）格式，放入你的项目目录中。

> 试卷可以是拍照截图、PDF 文档等，可借助豆包、AI PDF 等工具将图片/PDF 转换为 Markdown 格式，再提交给本技能。

### 3. 调用技能

在 AI IDE 中打开项目，输入提示词调用本技能：

```
@zero-to-pass 帮我复习 试卷文件.md
```

技能会自动逐题生成复习资料并写入 Markdown 文件中。
