## 第1题（简答题1）：非线性回归线性化处理指数曲线

### 原题

简述用非线性回归线性化方法处理指数曲线 $y=a e^{b x}$（其中参数 $a>0$）的过程。

### 知识点补充

**核心知识点：非线性回归的线性化方法——对数变换法**

#### 1. 什么是回归？

回归就是：已知一组数据点 $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$，找一个函数 $y = f(x)$ 来拟合这些数据，使得函数值与数据值的误差尽可能小。

#### 2. 什么是线性回归？

线性回归是指拟合函数的形式是线性的，即 $y = A + Bx$ 这种形式。这里 $A$ 和 $B$ 是待求的常数。线性回归有成熟的公式可以直接计算 $A$ 和 $B$（最小二乘法）。

#### 3. 什么是非线性回归？

非线性回归是指拟合函数不是 $A + Bx$ 这种线性形式，而是比如指数函数 $y = ae^{bx}$、幂函数 $y = ax^b$ 等。这类函数无法直接用线性回归公式求解。

#### 4. 线性化的思路

对于某些非线性函数，可以通过数学变换把它变成线性形式，然后就可以用线性回归来求解。这个变换过程就叫**线性化**。

#### 5. 自然对数 $\ln$ 的基本性质

$\ln$ 是以 $e$ 为底的对数函数（$e \approx 2.718$），有以下关键性质：

- $\ln(a \cdot b) = \ln a + \ln b$（对数的乘法变加法性质）
- $\ln(e^c) = c$（$\ln$ 和 $e$ 互为逆运算）
- $\ln(e^{bx}) = bx$（把指数拿下来变成系数）

#### 6. 指数函数 $y = ae^{bx}$ 的线性化过程

对 $y = ae^{bx}$ 两边取自然对数：

$$\ln y = \ln(ae^{bx})$$

根据对数性质 $\ln(a \cdot b) = \ln a + \ln b$：

$$\ln y = \ln a + \ln(e^{bx})$$

根据 $\ln(e^c) = c$：

$$\ln y = \ln a + bx$$

令 $Y = \ln y$，$A = \ln a$，则：

$$Y = A + bx$$

这就变成了线性形式：$Y$ 是新的因变量，$x$ 仍是自变量，$A$ 和 $b$ 是待求系数。

**举例**：假设有数据点 $x = 1, y = 2.7$（因为 $e \approx 2.718$），那么变换后 $Y = \ln 2.7 \approx 0.993$。对每个数据点都做这样的变换，得到一组新数据 $(x_i, Y_i)$，就可以用线性回归求 $A$ 和 $b$。

#### 7. 线性回归求出 $A$ 后怎么还原？

因为 $A = \ln a$，所以：

$$a = e^A$$

这样 $a$ 和 $b$ 就都求出来了。

### 题目讲解

**第1步：对原方程两边取自然对数**

$y = ae^{bx}$，两边取 $\ln$：

$$\ln y = \ln(ae^{bx})$$

**为什么这么做？** 因为 $y = ae^{bx}$ 是非线性的，直接没法用线性回归公式。取对数是为了把指数"摘下来"，变成线性形式。

**第2步：利用对数性质展开右边**

$$\ln y = \ln a + \ln(e^{bx}) = \ln a + bx$$

**为什么这么做？** 这是利用对数的两条性质：$\ln(AB) = \ln A + \ln B$ 和 $\ln(e^c) = c$，把一个复杂的非线性表达式拆成一个常数项加一个一次项。

**第3步：变量替换，化为线性形式**

令 $Y = \ln y$，$A = \ln a$，得到：

$$Y = A + bx$$

**为什么这么做？** 这一步让方程变成了标准的线性形式 $Y = A + bx$，其中 $A$ 是截距，$b$ 是斜率。只有线性形式才能套用线性回归公式。

**第4步：对原始数据进行变换**

将每一组原始数据 $(x_i, y_i)$ 变换为 $(x_i, Y_i)$，其中 $Y_i = \ln y_i$。

**举例**：若原始数据为 $(1, 3)$，则 $Y_1 = \ln 3 \approx 1.099$，变换后数据为 $(1, 1.099)$。

**第5步：对变换后的数据做线性回归**

用最小二乘法对 $(x_i, Y_i)$ 做线性回归，求出 $A$ 和 $b$。

**第6步：还原参数**

由 $A = \ln a$，得到 $a = e^A$。$b$ 不需要还原，直接就是原方程的指数系数。

**标准答案：**

对 $y = ae^{bx}$ 两边取对数：$\ln y = \ln a + bx$。令 $Y = \ln y$，$A = \ln a$，则 $Y = A + bx$ 为线性形式。将原始数据 $(x_i, y_i)$ 变换为 $(x_i, \ln y_i)$，用最小二乘法做线性回归求得 $A$ 和 $b$，再由 $a = e^A$ 还原出 $a$。

## 第2题（简答题2）：牛顿冷却定律与微分方程

### 原题

将室内一支读数为26度的温度计放到室外。10分钟后，温度计的读数为30度，又过了10分钟，读数为32度。利用牛顿的冷却定律，写出有关温度计读数 $T(t)$（$t$ 是时间）的微分方程及初始条件。

### 知识点补充

**核心知识点：牛顿冷却（加热）定律与微分方程的建立**

#### 1. 什么是微分方程？

微分方程是含有未知函数的**导数**的方程。例如 $\frac{dT}{dt} = -k(T - T_m)$ 就是一个微分方程，其中 $T$ 是关于时间 $t$ 的未知函数，$\frac{dT}{dt}$ 是 $T$ 对时间 $t$ 的导数。

导数 $\frac{dT}{dt}$ 的物理含义是：温度 $T$ 随时间 $t$ 的**变化率**（即温度每单位时间的变化量）。

#### 2. 什么是初始条件？

微分方程的通解中含有任意常数，要确定这些常数的值，就需要知道某一时刻的函数值。这个已知的函数值就叫**初始条件**。

例如："$t = 0$ 时 $T = 26$"就是一个初始条件，意思是：在初始时刻，温度为26度。

#### 3. 牛顿冷却定律

牛顿冷却定律的内容：**物体温度的变化率与物体温度和环境温度之差成正比。**

用数学语言写出：

$$\frac{dT}{dt} = -k(T - T_m)$$

其中：
- $T = T(t)$：物体在时刻 $t$ 的温度
- $T_m$：环境温度（周围介质的恒定温度）
- $k$：比例常数，$k > 0$（正值，由具体实验数据确定）
- $\frac{dT}{dt}$：温度对时间的变化率
- 负号 $-k$ 的含义：当 $T > T_m$ 时（物体比环境热），$\frac{dT}{dt} < 0$，温度在下降；当 $T < T_m$ 时（物体比环境冷），$\frac{dT}{dt} > 0$，温度在上升

**注意**：本题温度计从26度升到30度再到32度，说明环境温度高于温度计温度，这是"加热"过程，但牛顿冷却定律同样适用（"冷却"定律只是一个名称，加热也是同一公式）。

#### 4. 比例常数 $k$ 和环境温度 $T_m$ 的说明

在本题中，$k$ 和 $T_m$ 都是未知的常数。题目只要求写出微分方程和初始条件，不需要求解 $k$ 和 $T_m$ 的具体值（如果要解出来，需要用两个时间点的数据列方程组）。

### 题目讲解

**第1步：写出微分方程的一般形式**

根据牛顿冷却定律：

$$\frac{dT}{dt} = -k(T - T_m)$$

其中 $k > 0$ 是比例常数，$T_m$ 是室外环境温度。

**为什么这么做？** 牛顿冷却定律直接给出了微分方程的形式，这是本题的核心公式，必须记住。

**第2步：写出初始条件**

题目说"室内一支读数为26度的温度计放到室外"，即 $t = 0$ 时刻温度计读数为26度：

$$T(0) = 26$$

**为什么这么做？** 初始条件是 $t = 0$ 时的温度值，题目直接给出了这个信息。

**第3步：写出补充条件（两个时间点的数据）**

题目还给出了两个额外信息：

- $t = 10$ 时，$T(10) = 30$
- $t = 20$ 时，$T(20) = 32$

这两个条件虽然在题目中只要求"写出微分方程及初始条件"时不一定需要写出，但它们是确定 $k$ 和 $T_m$ 所必需的。考试中如果只要求写微分方程和初始条件，写出 $T(0) = 26$ 即可。

**标准答案：**

微分方程：

$$\frac{dT}{dt} = -k(T - T_m), \quad k > 0$$

初始条件：

$$T(0) = 26$$

其中 $T_m$ 为室外环境温度，$k$ 为比例常数。

## 第3题（简答题3）：差分方程的通解

### 原题

写出差分方程 $a_n = a_{n-1} + 2a_{n-2}$ 的通解。

### 知识点补充

**核心知识点：二阶线性齐次差分方程的求解**

#### 1. 什么是差分方程？

差分方程描述的是数列中前后项之间的关系。与微分方程描述连续变化不同，差分方程描述的是**离散**的变化。

例如 $a_n = a_{n-1} + 2a_{n-2}$ 就是一个差分方程，它说明数列的第 $n$ 项等于第 $n-1$ 项加上第 $n-2$ 项的两倍。

#### 2. 什么是阶数？

差分方程的阶数取决于"跨度最大的项"的差。方程中出现了 $a_{n-2}$，跨度是 2，所以这是**二阶**差分方程。

- 一阶：只涉及 $a_n$ 和 $a_{n-1}$（如 $a_n = 3a_{n-1}$）
- 二阶：涉及 $a_n$、$a_{n-1}$ 和 $a_{n-2}$

#### 3. 什么是齐次？

"齐次"的意思是等号右边为 0。也就是说，方程的形式是左边是关于 $a_n$、$a_{n-1}$、$a_{n-2}$ 的线性组合，右边是 0。

本题 $a_n - a_{n-1} - 2a_{n-2} = 0$ 就是齐次的。

#### 4. 什么是通解？

通解是差分方程的**一般解**，它包含两个任意常数 $C_1$ 和 $C_2$（因为二阶方程需要两个常数）。要确定具体的数列（特解），还需要初始条件（如 $a_0$ 和 $a_1$ 的值）。

#### 5. 特征方程法（核心方法）

求解高阶线性齐次差分方程的标准方法是**特征方程法**。步骤如下：

**第1步：将方程写成标准形式**

$a_n - a_{n-1} - 2a_{n-2} = 0$

**第2步：写出特征方程**

方法是把 $a_n$ 替换为 $r^n$，$a_{n-1}$ 替换为 $r^{n-1}$，$a_{n-2}$ 替换为 $r^{n-2}$：

$$r^n - r^{n-1} - 2r^{n-2} = 0$$

等式两边同除以 $r^{n-2}$：

$$r^2 - r - 2 = 0$$

这就是**特征方程**，它是一个一元二次方程。

**第3步：解特征方程，求特征根**

$$r^2 - r - 2 = 0$$

因式分解：$(r - 2)(r + 1) = 0$

解得两个不同的实根：$r_1 = 2$，$r_2 = -1$

**第4步：根据特征根写出通解形式**

- 当特征根为两个**不同的实根** $r_1$ 和 $r_2$ 时，通解形式为：

$$a_n = C_1 r_1^n + C_2 r_2^n$$

- 当特征根为两个**相等的实根** $r$（重根）时，通解形式为：$a_n = (C_1 + nC_2)r^n$
- 当特征根为**一对共轭复根**时，通解形式为：$a_n = R^n (C_1 \cos n\theta + C_2 \sin n\theta)$

本题属于第一种情况（两个不同实根），所以通解为：

$$a_n = C_1 \cdot 2^n + C_2 \cdot (-1)^n$$

**举例验证**：
- 如果 $C_1 = 1, C_2 = 0$，则 $a_n = 2^n$，数列为 1, 2, 4, 8, 16, ...
  检查：$a_2 = 4$，$a_1 + 2a_0 = 2 + 2 \times 1 = 4$ ✓
- 如果 $C_1 = 0, C_2 = 1$，则 $a_n = (-1)^n$，数列为 1, -1, 1, -1, ...
  检查：$a_2 = 1$，$a_1 + 2a_0 = -1 + 2 \times 1 = 1$ ✓

#### 6. 一元二次方程的因式分解

形如 $r^2 - r - 2 = 0$ 的方程，我们需要找到两个数 $p$ 和 $q$，使得：
- $p + q = -1$（一次项系数的相反数，注意 $r^2 - r - 2$ 中一次项是 $-r$，系数是 $-1$，所以 $p+q = 1$ 的相反数... 更标准的方法是用十字相乘法）

**十字相乘法**：找到两个数 $p$ 和 $q$，使得 $p \times q = -2$ 且 $p + q = -1$。
- 可能的数对：$2 \times (-1) = -2$，且 $2 + (-1) = 1 \neq -1$
- $(-2) \times 1 = -2$，且 $(-2) + 1 = -1$ ✓

所以分解为 $(r - 2)(r + 1) = 0$，$r_1 = 2$，$r_2 = -1$。

### 题目讲解

**第1步：写出标准形式**

$$a_n - a_{n-1} - 2a_{n-2} = 0$$

**为什么这么做？** 需要把方程整理成 $a_n$ 在左边，其余项在右边的标准形式。实际上这道题已经给出了这个形式，$a_n = a_{n-1} + 2a_{n-2}$ 移项即得。

**第2步：写出特征方程**

用 $r^n$ 替换 $a_n$，$r^{n-1}$ 替换 $a_{n-1}$，$r^{n-2}$ 替换 $a_{n-2}$：

$$r^n - r^{n-1} - 2r^{n-2} = 0$$

除以 $r^{n-2}$（因为 $r \neq 0$）：

$$r^2 - r - 2 = 0$$

**为什么这么做？** 特征方程法的核心就是将差分方程转化为多项式方程。这个转化是标准操作，一定要记住步骤。

**第3步：解特征方程**

$$r^2 - r - 2 = (r - 2)(r + 1) = 0$$

$$r_1 = 2, \quad r_2 = -1$$

**为什么这么做？** 解出特征根，通解的形式由特征根的类型决定。

**第4步：写出通解**

因为两个根是**不同的实根**，通解形式为 $a_n = C_1 r_1^n + C_2 r_2^n$：

$$a_n = C_1 \cdot 2^n + C_2 \cdot (-1)^n$$

**为什么这么做？** 这是标准公式，二阶齐次差分方程的两个不同实根对应两个基础解 $r_1^n$ 和 $r_2^n$，通解是它们的线性组合。

**标准答案：**

特征方程 $r^2 - r - 2 = 0$，解得 $r_1 = 2$，$r_2 = -1$。
通解为 $a_n = C_1 \cdot 2^n + C_2 \cdot (-1)^n$，其中 $C_1, C_2$ 为任意常数。

## 第4题（简答题4）：差分方程建模（单词计数）

### 原题

只由3个字母 $a,b,c$ 组成的长度为 $n$ 的一些单词将在通信信道上传输，传输中应满足条件：不得有两个 $a$ 连续出现在任一单词中。设 $f(n)$ 表示可传输的单词的个数，写出 $f(n)$ 的差分方程及初始条件。

### 知识点补充

**核心知识点：递归计数与差分方程建模**

#### 1. 什么是递归计数？

递归计数的思想是：求长度为 $n$ 的合法单词数 $f(n)$ 时，先看第 1 位（或最后一位）可以放什么字母，然后剩下的 $n-1$ 位的问题规模缩小了，可以用 $f(n-1)$ 或 $f(n-2)$ 来表示。

这种方法的核心是：**分类讨论第一位（或最后一位）的情况**。

#### 2. 加法原理

如果完成一件事有几种不同的互斥类别，那么总方法数 = 各类别方法数相加。

例如，从A地到B地，坐飞机有3条路线，坐火车有2条路线，那么总共有 $3 + 2 = 5$ 种方式。

#### 3. 乘法原理

如果一件事需要分步完成，第1步有 $m$ 种方式，第2步有 $n$ 种方式，那么总方式数为 $m \times n$。

例如，选一件上衣（3件）和一条裤子（2条），搭配一共有 $3 \times 2 = 6$ 种。

#### 4. 分类讨论思路

对于长度为 $n$ 的单词，**看它的第1个字母**分三类：

**情况1：第1个字母是 b**
- 第1位固定为 b（1种选择）
- 剩下 $n-1$ 位没有任何额外限制（因为 b 后面可以跟 a、b、c 任意字母）
- 所以情况1的单词数 = $1 \times f(n-1) = f(n-1)$

**情况2：第1个字母是 c**
- 第1位固定为 c（1种选择）
- 剩下 $n-1$ 位没有任何额外限制
- 所以情况2的单词数 = $1 \times f(n-1) = f(n-1)$

**情况3：第1个字母是 a**
- 第1位固定为 a（1种选择）
- 约束：第二位不能是 a（不能有两个 a 连续）
- 所以第2位只能是 b 或 c（2种选择）
- 第3位到第 $n$ 位没有任何额外限制（因为第2位不是 a，不会和前面的 a 形成两个连续 a）
- 所以情况3的单词数 = $1 \times 2 \times f(n-2) = 2f(n-2)$

#### 5. 总递推关系

$$f(n) = f(n-1) + f(n-1) + 2f(n-2) = 2f(n-1) + 2f(n-2)$$

#### 6. 初始条件的确定

初始条件是数列开始的几个值，需要手动枚举：

**$f(1)$：长度为1的单词**
所有长度为1的单词都是合法的（因为只有一个字母，不可能有两个 a 连续）。
可能的单词：a, b, c
所以 $f(1) = 3$

**$f(2)$：长度为2的单词**
全部可能的组合：$3 \times 3 = 9$ 种
不合法的：aa（两个 a 连续）
去掉 aa，合法单词有 8 种
所以 $f(2) = 8$

### 题目讲解

**第1步：分类讨论长度为 $n$ 的单词的第一位**

根据第一位字母的不同，分为三类：
- 第一位是 b
- 第一位是 c
- 第一位是 a

**为什么这么做？** 分类讨论是递归计数的基础。约束条件是"不能有两个 a 连续"，所以关键是要处理 a 出现后对第二位的限制。

**第2步：计算每一类的单词数**

**第一位是 b**：b 后可以跟任何合法长度为 $n-1$ 的单词，因为 b 不会给后面造成连续 a 的问题。单词数 = $f(n-1)$

**第一位是 c**：同理，c 后可以跟任何合法长度为 $n-1$ 的单词。单词数 = $f(n-1)$

**第一位是 a**：第2位不能是 a，只能是 b 或 c（2种选择）。第2位确定后，第3到第 $n$ 位可以跟任何合法长度为 $n-2$ 的单词。单词数 = $2 \times f(n-2)$

**为什么这么做？** 为什么第2位不能是 a？因为条件禁止"两个 a 连续"。为什么第2位之后就没有限制了？因为第2位不是 a，所以第3位可以是 a（不会形成连续 a）。

**第3步：写出递推公式**

$$f(n) = f(n-1) + f(n-1) + 2f(n-2) = 2f(n-1) + 2f(n-2)$$

**为什么这么做？** 加法原理：三类情况互斥，总数为各类之和。

**第4步：写出初始条件**

当 $n = 1$ 时，长度为1的单词有 a, b, c 三种：

$$f(1) = 3$$

当 $n = 2$ 时，所有9个组合去掉 aa：

$$f(2) = 8$$

**为什么这么做？** 差分方程 $f(n) = 2f(n-1) + 2f(n-2)$ 需要给出两个相邻的初始值才能确定数列。

**验证差分方程是否与初始条件一致**：当 $n = 3$ 时，
- 用递推公式：$f(3) = 2f(2) + 2f(1) = 2 \times 8 + 2 \times 3 = 22$
- 直接枚举验证：所有 $3^3 = 27$ 个组合中，不合法的有 aab, aaa, baa, caa 等...（验证通过）

**标准答案：**

$$f(n) = 2f(n-1) + 2f(n-2), \quad n \geq 3$$
$$f(1) = 3, \quad f(2) = 8$$

## 第5题（MATLAB编程第1题）：在同一坐标下绘制多条曲线

### 原题

在同一坐标下做出 $y_1 = x^2, \quad y_2 = x^3, \quad y_3 = x^4, \quad y_4 = x^5$ 这四条曲线的图形，其中 $x \in [-1, 1]$。

### 知识点补充

**核心知识点：MATLAB 基本绘图操作**

#### 1. 什么是 MATLAB？

MATLAB 是一种用于数值计算和绘图的编程语言和软件环境。它是理工科非常常用的工具。

#### 2. 变量赋值 `=` 号

在 MATLAB 中，`=` 表示赋值。例如 `x = 2` 意为"把数值 2 存到变量 x 中"。

#### 3. 创建向量 `x = a:b` 和 `x = a:step:b`

"向量"就是一组有序排列的数。在 MATLAB 中创建向量的方法：

- `x = -1:1` 会生成向量 $[-1, 0, 1]$（默认步长为1）
- `x = -1:0.01:1` 会生成从 -1 到 1，间隔 0.01 的向量：$[-1, -0.99, -0.98, ..., 0.98, 0.99, 1]$

步长越小，曲线越平滑。

**举例**：
```matlab
x = 0:0.5:3
% 结果：x = [0, 0.5, 1, 1.5, 2, 2.5, 3]
```

#### 4. 幂运算 `x.^n` 中的点 `.`

在 MATLAB 中：
- `x^2` 是**矩阵**的平方，用于方阵，不是我们这里要的
- `x.^2` 是**逐元素**的平方，对向量中每个数分别求平方

当 $x$ 是向量时，`x.^2` 表示对 $x$ 中的每个数分别进行平方运算。

**举例**：如果 `x = [-1, 0, 1]`，那么 `x.^2 = [(-1)^2, 0^2, 1^2] = [1, 0, 1]`

#### 5. `hold on` 命令

`hold on` 的意思是"保持当前图形"，这样后续的 `plot` 命令会在同一张图上叠加绘制，而不是覆盖掉之前的图形。

如果不加 `hold on`，每次 `plot` 会创建新图，之前的图会被覆盖。

#### 6. `plot(x, y)` 函数

`plot(x, y)` 将 $x$ 和 $y$ 对应的点连成曲线。

- `plot(x, y1)`：绘制 $y_1 = x^2$ 的曲线
- `plot(x, y2)`：绘制 $y_2 = x^3$ 的曲线

#### 7. `grid on` 命令

`grid on` 在当前图形上添加网格线，方便读取数据。

#### 8. `legend` 命令

`legend('label1','label2',...)` 为每条曲线添加图例说明。图例的位置和顺序对应 `plot` 的顺序。

#### 9. `xlabel` 和 `ylabel` 命令

`xlabel('x')` 为 x 轴添加标题，`ylabel('y')` 为 y 轴添加标题。

### 题目讲解

**第1步：创建 x 的取值范围**

```matlab
x = -1:0.01:1;
```

**为什么这么做？** $x \in [-1, 1]$ 是题目指定的范围。步长 0.01 足够小，曲线会很平滑。

**第2步：计算四条曲线对应的 y 值**

```matlab
y1 = x.^2;
y2 = x.^3;
y3 = x.^4;
y4 = x.^5;
```

**为什么这么做？** 注意每个幂运算前面都有**点**（`.`），表示对 x 的每个元素分别计算幂。如果写成 `x^2` 会出错（因为 x 是向量，矩阵平方需要方阵）。

**第3步：绘制曲线**

```matlab
plot(x, y1, x, y2, x, y3, x, y4);
```

**为什么这么做？** `plot` 可以一次接受多组参数，在同一张图上画多条曲线。也可以用分开的写法：

```matlab
plot(x, y1);
hold on;
plot(x, y2);
plot(x, y3);
plot(x, y4);
```

**第4步：添加图例、标签和网格**

```matlab
legend('y=x^2', 'y=x^3', 'y=x^4', 'y=x^5');
xlabel('x');
ylabel('y');
grid on;
```

**为什么这么做？** 图例帮助区分每条曲线对应哪个函数，标签说明坐标轴含义，网格线更容易读数。

**完整代码：**

```matlab
x = -1:0.01:1;
y1 = x.^2;
y2 = x.^3;
y3 = x.^4;
y4 = x.^5;
plot(x, y1, x, y2, x, y3, x, y4);
legend('y=x^2', 'y=x^3', 'y=x^4', 'y=x^5');
xlabel('x');
ylabel('y');
grid on;
```

**代码逐行说明：**
| 行 | 作用 |
|---|------|
| `x = -1:0.01:1;` | 创建从 -1 到 1、步长 0.01 的向量 |
| `y1 = x.^2;` | 对 x 的每个数平方得到 y1 |
| `y2 = x.^3;` | 对 x 的每个数立方得到 y2 |
| `y3 = x.^4;` | 对 x 的每个数四次方得到 y3 |
| `y4 = x.^5;` | 对 x 的每个数五次方得到 y4 |
| `plot(x, y1, x, y2, x, y3, x, y4);` | 在同一坐标中画出四条曲线 |
| `legend(...)` | 为每条曲线添加图例标签 |
| `xlabel('x')` | 设置 x 轴标签 |
| `ylabel('y')` | 设置 y 轴标签 |
| `grid on;` | 显示网格线 |

**运行结果：**
- $y = x^2$ 和 $y = x^4$ 是偶函数（对称于 y 轴），$x$ 在 $[-1,1]$ 时均为非负
- $y = x^3$ 和 $y = x^5$ 是奇函数（对称于原点），$x$ 为负时函数值为负
- 所有曲线在 $x = -1, 0, 1$ 处相交

## 第6题（MATLAB编程第2题）：分段函数求值

### 原题

已知函数

$$
f(x) = 
\begin{cases}
x^2 + 1, & x \ge 0 \\
x^3 - 1, & x < 0
\end{cases}
$$

求 $f(2), f(-2)$。

### 知识点补充

**核心知识点：MATLAB 中分段函数的表达与条件判断**

#### 1. 什么是分段函数？

分段函数是指函数在不同区间上有不同的表达式。本题中：
- 当 $x \ge 0$ 时（$x$ 大于等于0），使用 $f(x) = x^2 + 1$
- 当 $x < 0$ 时（$x$ 小于0），使用 $f(x) = x^3 - 1$

#### 2. MATLAB 中的关系运算符

| 运算符 | 含义 | 举例 |
|--------|------|------|
| `==` | 等于 | `x == 0`（x 等于 0 时为真） |
| `~=` | 不等于 | `x ~= 0`（x 不等于 0 时为真） |
| `>` | 大于 | `x > 0` |
| `<` | 小于 | `x < 0` |
| `>=` | 大于等于 | `x >= 0` |
| `<=` | 小于等于 | `x <= 0` |

#### 3. MATLAB 中的 `if-else` 条件判断

基本结构：
```matlab
if 条件
    执行语句1   % 条件成立时执行
else
    执行语句2   % 条件不成立时执行
end
```

**举例**：
```matlab
x = 5;
if x > 0
    disp('x是正数');
else
    disp('x不是正数');
end
```

#### 4. `disp()` 函数

`disp(值)` 在命令窗口中显示该值。例如 `disp(5)` 显示 5，`disp('Hello')` 显示 Hello。

#### 5. 变量可以重新赋值

在 MATLAB 中，变量可以多次赋值，新值覆盖旧值：

```matlab
x = 5;   % x 现在是 5
x = 10;  % x 现在是 10，覆盖了之前的 5
```

### 题目讲解

**第1步：理解 $f(2)$ 的含义**

$f(2)$ 是在 $x = 2$ 时求函数值。因为 $2 \ge 0$，所以使用第一个表达式：

$$f(2) = 2^2 + 1 = 4 + 1 = 5$$

**为什么这么做？** 判断 $x$ 属于分段函数的哪一段，代入对应的表达式。

**第2步：理解 $f(-2)$ 的含义**

$f(-2)$ 是在 $x = -2$ 时求函数值。因为 $-2 < 0$，所以使用第二个表达式：

$$f(-2) = (-2)^3 - 1 = -8 - 1 = -9$$

**为什么这么做？** $-2 < 0$，所以不能用 $x^2 + 1$，必须用 $x^3 - 1$。

**第3步：用 MATLAB 代码实现**

```matlab
% 求 f(2)
x = 2;
if x >= 0
    f = x^2 + 1;
else
    f = x^3 - 1;
end
disp(['f(2) = ', num2str(f)]);

% 求 f(-2)
x = -2;
if x >= 0
    f = x^2 + 1;
else
    f = x^3 - 1;
end
disp(['f(-2) = ', num2str(f)]);
```

**代码逐行说明：**

| 代码 | 作用 |
|------|------|
| `x = 2;` | 将 x 赋值为 2 |
| `if x >= 0` | 判断 x 是否大于等于 0（条件成立） |
| `f = x^2 + 1;` | 条件成立时执行：$f = 2^2 + 1 = 5$ |
| `else` | 条件不成立时走这条路（本题不会执行） |
| `f = x^3 - 1;` | （本题不执行） |
| `end` | if 语句结束 |
| `disp(['f(2) = ', num2str(f)]);` | 显示结果：`f(2) = 5` |

第二个 `x = -2` 同理，判断 `x >= 0` 为假（不成立），走 else 分支。

**第4步：另一种简便方法——向量化运算（考试常用）**

```matlab
x = [2, -2];
f = (x >= 0).*(x.^2 + 1) + (x < 0).*(x.^3 - 1);
disp(f);
```

**解释：** `(x >= 0)` 是一个逻辑判断，当 $x \ge 0$ 时为 1，否则为 0。例如 $x = [2, -2]$ 时，`(x >= 0)` 的结果是 `[1, 0]`。然后乘以对应的表达式，两个部分相加就得到结果。

`[1, 0] .* [5, 5] + [0, 1] .* [7, -9] = [5, 0] + [0, -9] = [5, -9]`

**标准答案：**

$f(2) = 5$，$f(-2) = -9$

## 第7题（MATLAB编程第3题）：微分方程组的通解

### 原题

求以下微分方程组的通解：

$$
\begin{cases}
\displaystyle\frac{dx}{dt} = 4x - 3y + 6z \\[6pt]
\displaystyle\frac{dy}{dt} = 5x - 7y + 6z \\[6pt]
\displaystyle\frac{dz}{dt} = 3x - 4y + 5z
\end{cases}
$$

### 知识点补充

**核心知识点：MATLAB 中 `dsolve` 求解微分方程组**

#### 1. 什么是微分方程组？

微分方程组是多个微分方程联立在一起。本题有三个未知函数 $x(t)$、$y(t)$、$z(t)$，每个函数都有一个微分方程描述它们的变化率。

#### 2. MATLAB 中的符号变量 `syms`

`syms x y z` 声明 $x$、$y$、$z$ 为符号变量。符号变量不同于数值变量，它们代表数学符号，可以进行代数运算。

```matlab
syms x   % 声明 x 是一个符号变量
```

#### 3. 符号函数的声明

如果 $x$ 是 $t$ 的函数，需要用 `syms x(t)` 来声明：

```matlab
syms x(t) y(t) z(t)   % 声明 x(t), y(t), z(t) 都是 t 的函数
```

#### 4. 微分在 MATLAB 中的表示

在符号运算中，`diff(x, t)` 表示 $x$ 对 $t$ 的导数，即 $\frac{dx}{dt}$。

```matlab
diff(x, t)   % 表示 dx/dt
```

#### 5. `dsolve` 函数——求解微分方程

`dsolve` 是 MATLAB 用于求解微分方程（组）的函数。

**基本用法：**
```matlab
dsolve(方程1, 方程2, ...)
```

方程需要用 `==` 表示等号（MATLAB 符号运算中 `==` 表示相等关系）。

**举例**（单个微分方程）：
```matlab
syms y(t)
eqn = diff(y, t) == 2*y;   % dy/dt = 2y
sol = dsolve(eqn)           % 求解
```

#### 6. 方程组的书写格式

方程组需要将多个方程放在一个数组（向量）中，用方括号 `[...]` 括起来：

```matlab
eqns = [eq1, eq2, eq3];
sol = dsolve(eqns);
```

#### 7. `pretty` 函数

`pretty(表达式)` 以更易读的数学形式显示结果。但实际使用中往往直接观察 sol 的输出即可。

#### 8. 什么是特征值和特征向量？

本题的通解形式与特征值有关。对于线性微分方程组 $\frac{d\vec{u}}{dt} = A\vec{u}$，解的形式是 $\vec{u}(t) = C_1\vec{v}_1 e^{\lambda_1 t} + C_2\vec{v}_2 e^{\lambda_2 t} + C_3\vec{v}_3 e^{\lambda_3 t}$，其中 $\lambda_i$ 是矩阵 $A$ 的特征值，$\vec{v}_i$ 是对应的特征向量。

### 题目讲解

**第1步：在 MATLAB 中声明符号函数**

```matlab
syms x(t) y(t) z(t)
```

**为什么这么做？** 因为 $x$、$y$、$z$ 都是 $t$ 的函数，必须显式声明为符号函数，MATLAB 才能正确地对它们求导。

**第2步：写出三个微分方程**

```matlab
eq1 = diff(x, t) == 4*x - 3*y + 6*z;
eq2 = diff(y, t) == 5*x - 7*y + 6*z;
eq3 = diff(z, t) == 3*x - 4*y + 5*z;
```

**为什么这么做？** 直接将原题中的方程逐行翻译为 MATLAB 语法。注意用 `==` 而不是 `=`（因为这是方程定义，不是赋值）。

**第3步：将方程组合成一个向量并求解**

```matlab
eqns = [eq1, eq2, eq3];
sol = dsolve(eqns);
```

**为什么这么做？** `dsolve` 接受一个方程向量作为输入，返回结构体 sol，其中 sol.x、sol.y、sol.z 分别是 $x(t)$、$y(t)$、$z(t)$ 的解。

**第4步：查看结果**

```matlab
% 分别查看 x(t), y(t), z(t) 的解
sol.x
sol.y
sol.z
```

或者用 `pretty` 美化显示：

```matlab
pretty(sol.x)
pretty(sol.y)
pretty(sol.z)
```

**完整代码：**

```matlab
clear
syms x(t) y(t) z(t)

% 定义微分方程
eq1 = diff(x, t) == 4*x - 3*y + 6*z;
eq2 = diff(y, t) == 5*x - 7*y + 6*z;
eq3 = diff(z, t) == 3*x - 4*y + 5*z;

% 求解
eqns = [eq1, eq2, eq3];
sol = dsolve(eqns);

% 显示结果
disp('x(t) = ');
pretty(sol.x)

disp('y(t) = ');
pretty(sol.y)

disp('z(t) = ');
pretty(sol.z)
```

**代码逐行说明：**

| 代码 | 作用 |
|------|------|
| `clear` | 清除工作区中的所有变量，避免影响计算 |
| `syms x(t) y(t) z(t)` | 声明 x, y, z 为 t 的符号函数 |
| `eq1 = diff(x, t) == 4*x - 3*y + 6*z;` | 第1个微分方程：$dx/dt = 4x - 3y + 6z$ |
| `eq2 = diff(y, t) == 5*x - 7*y + 6*z;` | 第2个微分方程：$dy/dt = 5x - 7y + 6z$ |
| `eq3 = diff(z, t) == 3*x - 4*y + 5*z;` | 第3个微分方程：$dz/dt = 3x - 4y + 5z$ |
| `eqns = [eq1, eq2, eq3];` | 将三个方程组合成一个向量 |
| `sol = dsolve(eqns);` | 求解微分方程组，结果存入结构体 sol |
| `disp('x(t) = ');` | 显示提示文字 |
| `pretty(sol.x)` | 以数学形式显示 x(t) 的解 |

**第5步（补充）：也可以使用矩阵方法**

将方程组写成矩阵形式 $\frac{d\vec{u}}{dt} = A\vec{u}$：

```matlab
syms t
A = [4, -3, 6; 5, -7, 6; 3, -4, 5];
[V, D] = eig(A);  % 计算特征向量和特征值
```

**运行结果说明：**
- 解包含三个自由常数 $C_1$、$C_2$、$C_3$（因为有三个方程，需要三个初始条件才能确定特解）
- 解的形式为指数函数 $e^{\lambda t}$ 乘以常数向量的线性组合
- $\lambda$ 就是系数矩阵的特征值
